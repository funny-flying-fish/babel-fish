PPTX → XLSX

Extract text from a marked PowerPoint file into an XLSX translation template.

📊
Drop .pptx file here or browse
Single file
  • Text is extracted by PowerPoint paragraph (a:p), not by visual line wrap.
  • Keys are based on Selection Pane names: shapeName + paragraphIndex.
  • Tables and other graphicFrame text are included when stored as editable DrawingML text.
  • Empty paragraphs are skipped.
  • Add translation columns to the XLSX, then use the Translations -> PPTX tab to generate localized presentations.
