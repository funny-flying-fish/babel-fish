Babel Fish — PPTX Converter

What does this tool do?

Prepares PowerPoint (.pptx) files with stable translation IDs, extracts text into an XLSX translation template, and injects translations back — producing a localized presentation.

  • Prepare PPTX IDs: add stable Selection Pane names to editable text containers
  • PPTX → XLSX: extract text by shapeName + paragraphIndex
  • XLSX → PPTX: inject translated text back into the marked file
  • Preserves formatting (font, size, color, bold)
  • Reports missing, duplicated, or moved IDs during conversion

Tab 1: Prepare PPTX IDs

Run this first. The tool writes stable translation IDs into the PowerPoint Selection Pane names stored in cNvPr name. Existing IDs such as tr_7f3a91_body are kept; duplicates are repaired.

1 Upload a .pptx file

Drag and drop an original or already marked PowerPoint file into the Prepare tab.

2 Click "Prepare PPTX IDs"

The tool marks text containers with names like tr_7f3a91_body. Re-run this step after duplicating slides to normalize copied IDs.

Tab 2: PPTX → XLSX

Extracts text from a marked PowerPoint presentation into a structured XLSX file ready for translation.

1 Upload a .pptx file

Drag and drop a PowerPoint file into the dropzone, or click to browse.

2 Set the source language

Enter the language code of the original text (default: EN). This becomes the column header in the XLSX.

3 Click "Extract text"

The tool parses every slide, extracts text from shapes, graphic frames, tables stored as editable DrawingML text, and PowerPoint paragraphs, then generates an XLSX file.

4 Download the XLSX and add translations

Open the downloaded file and add new columns for each target language (e.g. ES, FR, DE).

The real match is shapeName + paragraphIndex. slideNumber is informational: if a text box moves to another slide, the tool still replaces it by shapeName and shows a warning.

Tab 3: XLSX → PPTX

Takes a translated XLSX and the marked PPTX, and produces a new presentation with the translated text.

1 Upload the translated XLSX

Drop the XLSX file with translations into the left dropzone. The language dropdown will populate automatically from the header row.

2 Upload the original PPTX

Drop the marked .pptx file into the right dropzone. This serves as the template.

3 Select a language and convert

Pick the target language from the dropdown and click Convert. The tool replaces text in every matched position and generates a new .pptx file.

Tip: You can switch to a different language in the dropdown and click Convert again — no need to re-upload the files.

How formatting is preserved

The translated text is placed into the first text run of each paragraph. The formatting properties (font, size, color, bold, italic) of that first run are preserved. If the original paragraph had multiple runs with different formatting, only the first run's style is kept — subsequent runs are emptied.

Important: Do not change shapeName or paragraphIndex in the XLSX. If an ID cannot be matched, the tool skips that row and shows a warning report.

FAQ

Does it handle images, charts, or tables?

It handles editable text inside shapes (p:sp) and graphic frames (p:graphicFrame), including many tables. Images and chart labels that are not stored as editable DrawingML text are left unchanged.

What if I add or remove slides after extraction?

The main IDs are not tied to slide numbers. If slides are reordered, the tool matches by shapeName. If slides are duplicated, PowerPoint may copy the same IDs, so run Prepare PPTX IDs again before extracting or injecting.

Can I use this for partial translations?

Yes. Only cells with non-empty values in the selected language column will be replaced. Missing translations are left as-is.

How large can the PPTX be?

The tool runs entirely in the browser. Files up to ~100 MB work fine in modern browsers. Very large files may take a few seconds to process.

Сначала запустите этот шаг. Инструмент записывает стабильные ID перевода в имена объектов PowerPoint из Selection Pane, которые хранятся в cNvPr name. Уже существующие ID вроде tr_7f3a91_body сохраняются, дубли исправляются.

1 Загрузите .pptx файл

Перетащите исходный или уже размеченный PowerPoint-файл во вкладку Prepare.

2 Нажмите «Prepare PPTX IDs»

Инструмент разметит текстовые контейнеры именами вроде tr_7f3a91_body. После дублирования слайдов запустите этот шаг снова, чтобы исправить скопированные ID.

Вкладка 2: PPTX → XLSX

Извлекает текст из размеченной PowerPoint-презентации в структурированный XLSX-файл, готовый для перевода.

3 Нажмите «Extract text»

Инструмент разберёт каждый слайд, извлечёт текст из фигур, graphic frame, таблиц, сохранённых как редактируемый DrawingML-текст, и абзацев PowerPoint, затем сгенерирует XLSX-файл.

4 Скачайте XLSX и добавьте переводы

Откройте скачанный файл и добавьте новые колонки для каждого целевого языка (например, ES, FR, DE).

Вкладка 3: XLSX → PPTX

Берёт переведённый XLSX и размеченный PPTX и создаёт новую презентацию с переведённым текстом.

1 Загрузите переведённый XLSX

Перетащите XLSX-файл с переводами в левую зону загрузки. Выпадающий список языков заполнится автоматически из строки заголовка.

2 Загрузите оригинальный PPTX

Перетащите размеченный .pptx файл в правую зону загрузки. Он служит шаблоном.

3 Выберите язык и конвертируйте

Выберите целевой язык из выпадающего списка и нажмите Convert. Инструмент заменит текст в каждой совпавшей позиции и сгенерирует новый .pptx файл.

Совет: Можно выбрать другой язык в выпадающем списке и нажать Convert снова — файлы загружать заново не нужно.

Как сохраняется форматирование

Переведённый текст помещается в первый текстовый фрагмент (run) каждого абзаца. Свойства форматирования (шрифт, размер, цвет, жирность, курсив) этого фрагмента сохраняются. Если в оригинальном абзаце было несколько фрагментов с разным форматированием, сохраняется стиль только первого — остальные очищаются.

Важно: Не изменяйте shapeName и paragraphIndex в XLSX. Если ID не найден, инструмент пропустит эту строку и покажет отчёт предупреждений.

FAQ

Работает ли с изображениями, диаграммами и таблицами?

Инструмент работает с редактируемым текстом внутри фигур (p:sp) и graphic frame (p:graphicFrame), включая многие таблицы. Изображения и подписи диаграмм, которые не хранятся как редактируемый DrawingML-текст, остаются без изменений.

Что если добавить или удалить слайды после извлечения?

Основные ID не привязаны к номерам слайдов. Если слайды переставлены, инструмент сопоставляет текст по shapeName. Если слайды дублировались, PowerPoint может скопировать те же ID, поэтому перед извлечением или вставкой снова запустите Prepare PPTX IDs.

Можно ли использовать для частичного перевода?

Да. Заменяются только ячейки с непустыми значениями в выбранной языковой колонке. Отсутствующие переводы остаются как есть.

Какой максимальный размер PPTX?

Инструмент работает полностью в браузере. Файлы до ~100 МБ обрабатываются нормально в современных браузерах. Очень большие файлы могут обрабатываться несколько секунд.
