Babel Fish — PPTX Converter

Simple idea

This tool helps translate a PowerPoint file without manually copying text back and forth.

  • First, mark the PPTX so every text block has a stable ID.
  • Then export the text to a very simple XLSX file.
  • Add translation columns to the XLSX.
  • Load the marked PPTX and XLSX, adjust font sizes if needed, and download translated PPTX files.

1. Prepare PPTX IDs

Do this before exporting text. The tool adds stable names to text blocks inside the PPTX. These names are used later to put translations back into the right places.

1 Upload the original PPTX

Open the Prepare PPTX IDs tab, drop the presentation, and click Prepare PPTX IDs.

2 Use the marked PPTX from now on

Download the marked file. Use this marked file for export and final generation.

2. Export PPTX to XLSX

This creates a translation spreadsheet. The file is intentionally simple.

1 Upload the marked PPTX

Open PPTX → XLSX, upload the marked PPTX, choose the source language code, and click Extract text.

XLSX format

keyENPLDE
tr_7f3a91_body__p0Original textPolish textGerman text

Do not edit the key column. It tells the tool where each line belongs. Add translation columns next to the source column, for example PL, DE, FR.

3. Generate Translated PPTX Files

Open Translations -> PPTX. Load the marked PPTX and the translated XLSX. The table appears only after both files are loaded.

1 Check font-size groups

The table groups similar text styles. Each row has a number, language size fields, a usage count, and a style label.

2 Preview examples

Hover a style label to see a quick preview. Click the label to add that style card to the panel on the right. Use × on a card to remove it.

3 Adjust sizes if needed

You can edit individual size cells. You can also use the multiplier at the top of each language column to shrink all sizes for that language.

4 Save presets and download PPTX

Save preset downloads a JSON with all size settings. Load preset applies it later. At the bottom of the table, use buttons like ↓ PL to download a PPTX for that language.

Important: If you duplicate slides or heavily edit text blocks in PowerPoint, run Prepare PPTX IDs again before exporting a new XLSX.

Quick FAQ

Can I translate only one language?

Yes. Add only the column you need, for example PL, then download ↓ PL.

Can I leave some cells empty?

Yes. Empty translation cells are skipped.

What should I never change?

Do not change key. You can edit language text and add new language columns.

Babel Fish — PPTX Конвертер

Простая идея

Инструмент помогает перевести PowerPoint без ручного копирования текста туда-сюда.

  • Сначала размечаем PPTX, чтобы у каждого текстового блока был стабильный ID.
  • Потом выгружаем текст в простой XLSX.
  • В XLSX добавляем колонки с переводами.
  • Загружаем размеченный PPTX и XLSX, при необходимости правим размеры шрифта и скачиваем готовые PPTX по языкам.

1. Prepare PPTX IDs

Этот шаг нужно сделать перед выгрузкой текста. Он добавляет стабильные имена текстовым блокам внутри PPTX. Потом по этим именам перевод встанет обратно в правильные места.

1 Загрузите исходный PPTX

Откройте вкладку Prepare PPTX IDs, загрузите презентацию и нажмите Prepare PPTX IDs.

2 Дальше используйте размеченный PPTX

Скачайте файл с разметкой. Именно его используйте для выгрузки XLSX и для финальной сборки переводов.

2. PPTX → XLSX

На этом шаге получается таблица для перевода. Она специально сделана максимально простой.

1 Загрузите размеченный PPTX

Откройте PPTX → XLSX, загрузите размеченный PPTX, укажите код исходного языка и нажмите Extract text.

Формат XLSX

keyENPLDE
tr_7f3a91_body__p0Original textPolski tekstDeutscher Text

Колонку key не меняем. Она говорит инструменту, куда вернуть каждую строку. Рядом можно добавлять языковые колонки: PL, DE, FR и так далее.

3. Translations -> PPTX

Откройте Translations -> PPTX. Загрузите размеченный PPTX и XLSX с переводами. Таблица появится только после загрузки обоих файлов.

1 Посмотрите группы размеров

Таблица группирует похожие текстовые стили. У каждой строки есть номер, поля размеров по языкам, количество использований и название стиля.

2 Проверяйте примеры

Наведите курсор на название стиля, чтобы быстро увидеть примеры. Нажмите на название, чтобы добавить карточку этого стиля в правую панель. Кнопка × удаляет карточку из панели.

3 Подправьте размеры, если нужно

Можно менять отдельные ячейки с размером. Можно использовать multiplier сверху языковой колонки, чтобы уменьшить все размеры для выбранного языка сразу.

4 Сохраните пресет и скачайте PPTX

Save preset скачивает JSON со всеми настройками размеров. Load preset загружает его обратно. Внизу таблицы есть кнопки вроде ↓ PL — они скачивают PPTX для конкретного языка.

Важно: если вы дублировали слайды или сильно меняли текстовые блоки в PowerPoint, перед новой выгрузкой XLSX снова запустите Prepare PPTX IDs.

Короткий FAQ

Можно сделать только один язык?

Да. Добавьте только нужную колонку, например PL, и скачайте ↓ PL.

Можно оставлять ячейки перевода пустыми?

Да. Пустые ячейки пропускаются.

Что точно нельзя менять?

Не меняйте key. Текст в языковых колонках менять можно, новые языковые колонки добавлять можно.
